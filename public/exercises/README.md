# Exercise JSON files

Drop Penny's exercise JSON files into **this folder** (`public/exercises/`).
They are served at runtime from `/exercises/<id>.json` and rendered by
`<ExerciseEmbed exerciseId="<id>" />`.

Expected files (from the "exercises files session 1_3" set):

- `s1-ex1-four-quadrants.json`        ← embedded in Session 1, Exercise 1
- `s2-ex1-team-52-card-pickup.json`
- `s2-ex2-client-portfolio-matrix.json`
- `s3-ex1-tech-stack-calculator.json`
- `s3-ex2-ai-vendor-scorecard.json`
- `manifest.json`

Each file must keep its exact name. Once `s1-ex1-four-quadrants.json` is here,
the Four Quadrants exercise renders live on `/workshops/session-1`.
